Blog Post 9. Developing an Effective Style

Part 1: Which guidelines from Chapter 10 do you use on a regular basis? Provide specific examples with explanations.

There’s lots of useful suggestions on how to build an effective professional style in chapter 10.
“Find out what readers think” is one of the most important guidelines to follow for me. It is the same idea that we have been talking through the whole class. And I have to say reader-centered approach is what we are supposed to learn from the class. We want to make sure that the readers can understand the material and feel in common with we what we write. Resonance will make the article more convincing. This requires that we must find out what our readers want to know and what methods of communication they prefer. Also, we need to consider the context and background knowledge of readers because these two factors will influence how they perceive information very heavily. Learn more about our primary audience background makes communication efficient and persuasive. For example, our project is a mentor app then there are two main audience groups: mentors and mentees. But the direct client of the project is administrator of the system (a BME professor). Thus our writings may have different readers each time. Write about what each group want is the best way to improve effectiveness of communication.

Part 2: Examine Sample 3 and Sample 2. Read the two sample proposals’ Objectives and Solution sections.

  • Which SOW does a better job linking Solution and Objectives?
  • How does the better example achieve this link?
  • How well does the solution address the scope?

SOW in sample 3 did a better job linking solution and objectives. Both samples list the solutions corresponds to objectives/ However, when coming to details, sample 3 provides more details while sample 2 emphasizes more on abstract ideas.

In sample 3, it has formulas with definition of each elements in those formulas. Thus we can easily understand the relationship among all the elements. Also, sample 3 lists clear instructions that help to illustrate how each use case flows. By checking all the formulas and instructions, we can easily figure out the logic behind the project.

The solution addressed the scope of the project very well since it only talked about elements corresponds or relevant to the materials within project scope.

Part 3: Examine Sample 1, Sample 2, and Sample 3. Which Resources section is more effective? What makes it more effective?

Sample 2 has the most effective resource section. It uses two well-formated table to show the details. One of the table is for tem-supplied items, another one is for client-supplied items. Each table contains the requirement, reason, source and cost columns. The reason attribute only appears in sample 2. It helps readers to understand why the resources are needed and how they would be used to improve the persuasiveness and effectiveness of communication.
